# [closingdoors.nyc](https://closingdoors.nyc/)

A minimalist subway schedule viewer written in React.

## Building and running locally

1. `npm run build`
2. `npm start`: starts page at [http://localhost:3000](http://localhost:3000)
To use storybook:

1. `npm run storybook`
